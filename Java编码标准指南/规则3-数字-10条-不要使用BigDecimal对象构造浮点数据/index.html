规则3-数字-10条-不要使用BigDecimal对象构造浮点数据

2022年02月07日
Java编码标准指南

因为浮点数字总是不准确的， 通过 new BigDecimal(double) 获取到的数据总是不准确的。

  • 通过 BigDecimal.valueOf能准确获取到数值。
  • 通过字符串类型能够获取到准确的十进制数据。
